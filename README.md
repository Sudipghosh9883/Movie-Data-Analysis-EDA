## 📌 Project Overview

This project demonstrates **Exploratory Data Analysis (EDA)** on a movie dataset using Python.

The goal of this project is to clean the dataset, perform exploratory analysis, create meaningful visualizations, and discover insights from movie data.

This project helped strengthen my understanding of **data cleaning, preprocessing, visualization, and exploratory data analysis using Python.**

---

## 📂 Dataset

The dataset contains information about **9,800+ movies**, including:

- 🎬 Movie Title
- 📅 Release Year
- ⭐ Popularity
- 🗳 Vote Count
- 🌟 Vote Average
- 🎭 Genre

# 🔍 Key Insights

- Drama is the most frequent movie genre in the dataset.
- Spider-Man: No Way Home has the highest popularity score.
- Movies released around 2020 are the most common.
- The dataset contains more than **9,800 movies** across multiple genres.
